# Book Two — Chapter Sources

This folder contains the editable `.docx` source files for **Book Two — The Application** (chapters 21–40).

The compiled, print-ready version of Book Two lives at the repo root: [`Alpha-Omega-Book-Two-Standalone.pdf`](../Alpha-Omega-Book-Two-Standalone.pdf).

## Structure

Book Two is divided into four parts:

- **Part V — Cultural and Belief Systems** (Chapters 21–25)
- **Part VI — Enforcement and Surveillance** (Chapters 26–30)
- **Part VII — Technology and AI** (Chapters 31–35)
- **Part VIII — Repair and Scale** (Chapters 36–40)
