### [Framing Map Addendum v1](./Framing-Map-Addendum-v1.pdf)

A two-page addendum to the Master Orientation & Control Guide. Explains how the three coexisting structural framings of the Alpha-Omega Lens relate to each other:

1. **Spine** — the 10-section Input-Alpha-Omega artifact (Layer C governance / input spec)
2. **Book** — the 18-chapter Book One canonical artifact (Layer A canonical)
3. **Reader** — the Public Reader Release (Layer A public-facing distillation)

The map establishes the rule: **Spine → Book → Reader**. Changes flow from input spec, through the full book, to the public reader. Changes that originate downstream must be reflected upstream before they are considered canonical.
